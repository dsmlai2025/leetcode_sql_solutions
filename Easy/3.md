## ðŸ’» SQL Solution

```sql
SELECT employee_id FROM (
SELECT 
employee_id,
DENSE_RANK() OVER(ORDER BY salary DESC) AS dn_rnk
FROM employee ) t
WHERE t.dn_rnk = 2
ORDER BY employee_id
