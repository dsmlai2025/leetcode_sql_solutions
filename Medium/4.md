## ðŸ’» SQL Solution

```sql
SELECT Department, Employee, Salary FROM (
    SELECT
    d.name AS Department, e.name AS Employee, e.salary AS Salary,
    DENSE_RANK() OVER(PARTITION BY e.departmentId ORDER BY salary DESC) AS d_rnk
    FROM Employee e
    JOIN Department d
    ON e.departmentId = d.id
) t
WHERE t.d_rnk = 1
